# Space Travel

This project is a study exercise focused on building a space travel-themed application.

## Features

- Browse destinations
- View details of the planets
- Interactive navigation

## Technologies Used

### Front end

- React
- TypeScript
- Vite
- TailwindCSS
- PostCSS

### Back end

- Prisma
- Fastify
- MongoDB
- bcrypt
- jsonwebtoken

## Testing

- Insomnia
- Jest
